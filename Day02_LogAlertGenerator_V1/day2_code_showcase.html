This mini project is a Python-based log file scanner that detects suspicious entries like failed login attempts, unauthorized access, or root usage. Alerts are printed to the console and saved to a separate file.

üíª Python Code (`log_alert_generator_v1.py`)


alert_keywords = ["Failed password", "unauthorized", "Invalid user", "root login"]

with open("system.log", "r") as logfile, open("alerts_output.txt", "w") as alertfile:
    for line in logfile:
        for keyword in alert_keywords:
            if keyword.lower() in line.lower():
                alert_msg = f"[ALERT] Suspicious activity detected: {line.strip()}"
                print(alert_msg)
                alertfile.write(alert_msg + "\n")
                break

üìå Summary

This script scanned the system log file, detected keywords like "Failed password" or "unauthorized", and stored only the suspicious lines as alerts. It builds fundamental SOC skills: log analysis, keyword matching, and security automation.
